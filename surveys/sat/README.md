# SAT — Фокус-группа по подготовке к SAT

Закрытый опросник для отбора участников фокус-группы по продукту SAT-подготовки Global Generation.

**URL**: `/focus-group`
**Admin**: `/focus-group/admin`

---

## Поток

1. **Hero** — лендинг с логотипами вузов (Harvard, Yale, Princeton, Duke, Brown) и CTA
2. **NDA** — соглашение о конфиденциальности (закрытая фокус-группа по приглашению)
3. **5-шаговая форма** — анкета кандидата
4. **Успех** — подтверждение + уведомление о связи через Telegram

---

## Форма (5 шагов)

### Шаг 1: О вас
| Поле | Тип | Валидация |
|------|-----|-----------|
| Имя | Текст | Обязательно |
| Telegram | Текст | Обязательно, авто-префикс `@` |
| Возраст | Число | 14–25 |
| Город | Текст | Обязательно |
| Уровень образования | Кнопки | 9 класс / 10 класс / 11 класс / Gap Year / 1 курс |

### Шаг 2: SAT и подготовка
| Поле | Тип | Валидация |
|------|-----|-----------|
| Когда планируете сдавать | Радио | Обязательно |
| Сдавали ли SAT раньше | Да/Нет | Обязательно |
| Предыдущий балл | Число | Только если сдавали, 400–1600 |
| Часов в неделю | Радио | Обязательно |
| Ресурсы подготовки | Чекбоксы | Khan Academy, College Board, YouTube, Репетитор, Курсы, Учебники, Другое |

### Шаг 3: Опыт подготовки
| Поле | Тип | Валидация |
|------|-----|-----------|
| Что нравится в подготовке | Textarea | Обязательно, 200+ символов для макс. баллов |
| Что расстраивает | Textarea | Обязательно, 200+ символов для макс. баллов |

### Шаг 4: Готовность
| Поле | Тип | Валидация |
|------|-----|-----------|
| Мотивация | Textarea | Обязательно, 200+ символов для макс. баллов |
| Готовность к сессии | Радио | Точно да / Скорее да / Может быть / Не уверен |
| Доступные дни | Кнопки (7 дней) | Минимум 1 |
| Доступное время | Чекбоксы | Утро 9-12 / День 12-17 / Вечер 17-21 |

### Шаг 5: Подтверждение
| Поле | Тип | Валидация |
|------|-----|-----------|
| Согласие на обработку данных | Чекбокс | Обязательно |
| Согласие на запись | Чекбокс | Обязательно |
| Откуда узнали | Кнопки | Telegram / Друг / Соцсети / Поиск / Другое (опционально) |

## Скоринг

**11 категорий, 105 баллов максимум.**

| # | Категория | Макс | Логика |
|---|-----------|------|--------|
| 1 | Когда сдаёте SAT | 20 | Ближайшие 3 мес=20, уже сдал=18, 6 мес=15, 6-12 мес=8, не решил=3 |
| 2 | Часов в неделю | 15 | 3-7ч=15, 7-14ч=13, 14+ч=10, 1-3ч=8, <1ч=3 |
| 3 | Мотивация | 15 | По длине текста: >=200=15, >=100=12, >=50=8, >=20=5, <20=0 |
| 4 | Предыдущий балл | 10 | >=1400=10, >=1200=8, >=1000=6, >=800=4, иначе=2 (0 если не сдавал) |
| 5 | Готовность к сессии | 10 | Точно=10, скорее=7, может быть=4, не уверен=1 |
| 6 | Ресурсы подготовки | 10 | >=4 шт=10, >=3=8, >=2=5, 1=3, 0=0 |
| 7 | Сдавали раньше | 5 | Да=5, нет=0 |
| 8 | Что нравится | 5 | По длине: >=200=5, >=100=4, >=50=3, >=20=2, <20=0 |
| 9 | Что расстраивает | 5 | Аналогично |
| 10 | Доступные дни | 5 | >=5=5, >=3=4, >=1=2, 0=0 |
| 11 | Доступное время | 5 | >=3=5, >=2=4, >=1=2, 0=0 |

### Уровни

| Баллы | Уровень |
|-------|---------|
| >= 80 | Отличный кандидат |
| >= 55 | Хороший кандидат |
| >= 30 | Средний кандидат |
| < 30 | Низкий приоритет |
